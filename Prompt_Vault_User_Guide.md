## 1. 产品简介

### Prompt Vault 是什么？

**Prompt Vault** 是一款帮你**存储、管理、测试和优化 Prompt**的工具。

简单来说：它就像你的 Prompt 笔记本 + AI 测试场。你可以把平时写的好用的 Prompt 整理在这里，需要时一键复制使用，还能直接测试效果、对比不同版本的输出。

### 它能帮你做什么？

| 场景 | Prompt Vault 如何帮忙 |
|------|----------------------|
| Prompt 散落各处 | 统一存储在一个地方，不再丢失 |
| 好的 Prompt 难以复用 | 分类整理 + 一键复制，随时取用 |
| 调优 Prompt 靠"试错" | 内置测试功能，看得见的效果对比 |
| 想看看哪个版本更好 | 版本历史记录，支持回滚 |

## 2. 快速开始

### 2.1 注册账号

1. 打开 Prompt Vault 网站
2. 点击「注册」按钮
3. 输入邮箱和密码，或者使用 Google/GitHub 账号登录
4. 完成邮箱验证（如果需要）

> 💡 **小贴士**：注册后会有简单的引导流程，带你熟悉主要功能。

### 2.2 创建你的第一个 Prompt

**方法一：从零开始**

1. 点击顶部导航的 **「+ 新建」** 按钮（或按 `⌘N` / `Ctrl+N`）
2. 选择「空白 Prompt」
3. 填写标题（如「小红书种草文案」）
4. 在正文区域输入你的 Prompt 内容
5. 选择保存到哪个文件夹（可选）
6. 点击「保存」或按 `⌘S`

**方法二：从模板开始**

1. 点击 **「+ 新建」**
2. 选择「从模板」
3. 浏览模板市场，选择你需要的场景（如「营销文案」「代码生成」）
4. 点击「使用模板」，模板会克隆到你的个人库
5. 根据你的需求修改内容，保存

> 🎉 **完成！** 现在你已经拥有第一个 Prompt 了。

### 2.3 快速上手三件事

| 操作 | 怎么做 |
|------|--------|
| **复制使用** | 在 Prompt 列表页，点击卡片上的「复制」图标，或直接点击进入详情页复制 |
| **测试效果** | 进入 Prompt 详情页，右侧就是测试面板，输入内容点击运行即可 |
| **找到想要的 Prompt** | 按 `⌘K` / `Ctrl+K` 打开搜索，输入关键词 |

## 3. 核心功能使用教程

### 3.1 存储：创建和管理 Prompt

**创建 Prompt**

- 点击「+ 新建」或 `⌘N`
- 支持 Markdown 格式，可以加粗、列表、代码块
- 可添加描述，方便后续理解用途

**编辑 Prompt**

- 进入任意 Prompt 详情页，直接修改内容
- 支持实时预览（按 `⌘P` 切换）
- 修改后记得保存（`⌘S`）

**删除 Prompt**

- 在列表页：鼠标悬停卡片 → 点击「⋯」→ 选择「删除」
- 或在详情页点击「删除」按钮
- 删除后有 30 天恢复期，可以在回收站找到

**收藏与置顶**

- 点击卡片上的 ⭐ 图标收藏
- 收藏的 Prompt 会在左侧边栏「⭐ 收藏」中显示
- 常用的 Prompt 可以置顶，保持在列表最前面

---

### 3.2 分类：文件夹 + 标签

**文件夹管理**

- 左侧边栏「📂 文件夹」区域
- 点击「+」创建新文件夹
- 支持多级文件夹（如「编程 → API 开发」）
- 创建 Prompt 时选择文件夹，或拖拽移动

**标签系统**

- 每个 Prompt 可以添加多个标签
- 在详情页的「标签」区域点击「+ 添加」
- 标签颜色会自动匹配，也支持自定义
- 点击左侧边栏的标签，可快速筛选

> 💡 **技巧**：常用的分类维度可以是「用途」（写作、编程、营销）+ 「场景」（小红书、公众号、技术文档）

---

### 3.3 搜索：快速找到目标

**基础搜索**

- 按 `⌘K` / `Ctrl+K` 打开搜索框
- 输入关键词，支持标题和正文内容搜索
- 搜索结果中关键词会**高亮显示**

**高级筛选**

- 在搜索页可以展开筛选器
- 按文件夹、标签、日期范围、评分筛选
- 筛选条件可以组合使用

**搜索技巧**

- 搜索历史会自动保存，点击搜索框可以看到
- 热门搜索标签可快速点击

---

### 3.4 测试：验证 Prompt 效果

**进入测试页面**

- 方法一：在 Prompt 详情页，右侧直接显示测试面板
- 方法二：选择 Prompt 后点击「测试」按钮

**运行测试**

1. 选择 AI 模型（如 GPT-4o、Claude）
2. 在「输入测试内容」区域填入测试文本
3. 点击「运行」或按 `⌘Enter`
4. 等待 AI 返回结果

**配置参数（可选）**

- 点击模型旁边的「⚙️」可调整参数
- **Temperature**：控制随机性（0-2），值越低越稳定
- **Max Tokens**：最大输出长度

**保存测试结果**

- 测试结果会自动保存到历史记录
- 可以点击「保存版本」将当前 Prompt 保存为新版本
- 查看历史测试记录，了解效果变化

> ⚠️ **注意**：测试会调用 AI API，可能产生费用。请确保你的账户已配置 API Key。

---

### 3.5 版本管理：追踪历史，轻松回滚

**版本自动保存**

- 每次保存 Prompt 会自动创建新版本
- 最多保留 10 个历史版本

**查看版本历史**

- 在 Prompt 详情页，点击「📊 版本历史」
- 可以看到每个版本的修改时间和内容
- 点击任意版本可预览

**回滚到旧版本**

- 找到想要恢复的版本
- 点击「恢复此版本」
- 会创建为新的版本，不会覆盖当前版本

**A/B 测试对比（进阶）**

- 在测试页面切换到「A/B 对比」模式
- 左侧选择/编辑 Prompt A，右侧选择 Prompt B
- 输入相同测试内容，点击「同时运行 A 和 B」
- 并排查看输出差异，标记优胜者

## 4. 常见使用场景示例

### 场景一：内容创作者沉淀模板

> 小红书文案、公众号文章、邮件回复...每次都要重新写？

**操作步骤：**

1. 创建「营销文案」文件夹
2. 针对常用场景创建 Prompt（如「小红书种草文案」）
3. 设置变量：`{{产品名}}`、`{{特点}}`、`{{目标人群}}`
4. 需要使用时，一键复制并填入变量值

**效果：** 原来 10 分钟写的文案，现在 1 分钟搞定 ✨

---

### 场景二：开发者调优 API 文档 Prompt

> 生成的 API 文档总是缺斤少两，想优化但不知道改哪个部分有效。

**操作步骤：**

1. 找到或创建 API 文档生成 Prompt
2. 进入测试页面，输入一个真实的 API 例子
3. 运行查看输出，记录问题
4. 修改 Prompt，保存新版本
5. 用 A/B 测试对比新旧版本输出
6. 确认最优版本，标记为「生产版」

**效果：** 有数据有对比，优化不再靠感觉 📊

---

### 场景三：团队共享最佳实践

> 团队里每个人都在重复造轮子，想把好的 Prompt 沉淀下来。

**操作步骤：**

1. 创建团队空间（如果功能已上线）
2. 邀请团队成员加入
3. 把经过验证的优质 Prompt 贡献到团队库
4. 设置合适的权限（谁可以编辑，谁只能查看）
5. 定期 review 效果评分，优化团队最佳实践

**效果：** 团队效率提升，知识不再流失 👥

---

### 场景四：学习优秀 Prompt 案例

> 看到别人分享的好 Prompt，想据为己用。

**操作步骤：**

1. 在模板市场浏览热门模板
2. 找到感兴趣的，点击「预览」查看效果
3. 点击「使用」克隆到个人库
4. 根据自己的需求修改
5. 保存并测试效果

**效果：** 站在巨人的肩膀上，快速起步 🚀

## 6. 常见问题解答

### Q1: Prompt Vault 免费吗？

> 目前基础功能（创建、存储、分类、搜索、基础测试）是**免费**的。高级功能（如 A/B 测试、团队协作、模板市场）可能会在后续版本中设为付费功能。具体定价请关注官方通知。

### Q2: 测试功能需要付费吗？

> 测试功能调用 AI API，会消耗你的 API 配额。你需要在设置中配置自己的 API Key（如 OpenAI、Anthropic 等）。平台不收取额外费用，只收取 API 调用成本。

### Q3: 我的 Prompt 会泄露吗？

> 不会。你的 Prompt 默认是**私有**的，只有你自己可以看到。我们不会将你的内容用于任何 AI 训练或公开分享。

> 如果你主动将 Prompt 发布到社区或团队空间，则表示你授权他人查看。

### Q4: 如何导入/导出我的 Prompt？

> 支持 JSON 和 Markdown 格式批量导入导出。具体操作：进入「设置」→「数据管理」→「导入/导出」。

### Q5: 不小心删除了 Prompt 能恢复吗？

> 能。删除的 Prompt 会在「回收站」中保留 **30 天**，期间可以恢复。超过 30 天将被永久删除。

### Q6: 支持哪些 AI 模型？

> 目前支持主流模型，包括：
> - OpenAI: GPT-4o, GPT-4, GPT-3.5
> - Anthropic: Claude 3.5, Claude 3
> - Google: Gemini
> - 本地模型（需配置）
>
> 后续会陆续支持更多模型。

### Q7: 手机上能用吗？

> 支持响应式布局，手机浏览器可以访问。核心功能（查看、搜索、复制）可用，编辑和测试功能在移动端做了适配优化。

### Q8: 遇到问题如何反馈？

> - 在应用中点击「帮助」→「反馈问题」
> - 或发送邮件到 support@promptvault.com
> - 也欢迎加入官方 Discord/微信群与其他用户交流
