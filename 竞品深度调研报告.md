# Prompt Vault 竞品深度调研报告

**调研日期**: 2026-03-11  
**调研范围**: PromptPerfect、PromptBase、PromptLayer、AIPRM

---

## 一、核心功能对比

### 1.1 功能对比表

| 功能维度 | PromptPerfect | PromptBase | PromptLayer | AIPRM |
|---------|--------------|------------|-------------|-------|
| **定位** | Prompt优化/生成工具 | Prompt交易市场 | 企业级Prompt管理平台 | 浏览器扩展模板库 |
| **核心功能** | Prompt优化、生成、调试 | 买卖/分享Prompt | 版本的注册表、API管理、LLM可观测性 | Prompt模板库、Claude/ChatGPT扩展 |
| **支持模型** | GPT-4, ChatGPT, Midjourney, Stable Diffusion | ChatGPT, Midjourney, DALL-E, Stable Diffusion, Gemini | 多LLM提供商 | ChatGPT, Claude |
| **团队协作** | 基础 | 基础（市场交易） | 强（可视化编辑器、多角色协作） | 基础（私有模板） |
| **版本控制** | 无 | 无 | 强（自动版本记录） | 无 |
| **评估测试** | 基础 | 无 | 强（Evals, A/B测试, 回归测试） | 无 |
| **API/集成** | API访问 | 无 | 丰富的API和中间件 | 无 |
| **部署方式** | 云端 | 云端 | 云端/自托管 | 浏览器扩展 |

### 1.2 各竞品详细功能

#### PromptPerfect (Jina AI)
- **Prompt优化**: 10秒内自动优化Prompt
- **多模型支持**: GPT-4, ChatGPT, Midjourney, Stable Diffusion
- **交互式优化器**: 内置聊天机器人形式的优化助手
- **API服务**: 支持将优化后的Prompt部署为API服务
- **免费托管**: 提供免费的Prompt托管计划

#### PromptBase
- **Prompt市场**: 买卖双方交易平台
- **搜索/浏览**: 按模型、类别搜索Prompt
- **销售功能**: 卖家可设置价格（上限$4.99新卖家）
- **Favorites**: 收藏功能
- **评价系统**: 5星评分机制

#### PromptLayer
- **Prompt Registry**: 可视化Prompt注册表和管理
- **版本控制**: 自动记录每个LLM调用的版本
- **LLM可观测性**: 日志追踪、调试、回放
- **评估(Evals)**: 回归测试、A/B测试、批量运行
- **协作功能**: 非技术人员也可参与Prompt迭代
- **多提供商支持**: 统一的抽象层支持多LLM提供商
- **合规认证**: SOC2 Type 2, GDPR, HIPAA, CCPA

#### AIPRM
- **浏览器扩展**: Chrome扩展形式
- **模板库**: 社区Prompt模板 + 认证模板
- **私有模板**: 付费用户可保存私有Prompt
- **列表管理**: Favorites、Hidden、自定义列表
- **Tone/Style控制**: 调整输出语气和风格
- **Power Continue**: 扩展的续写功能

## 二、定价模式

### 2.1 定价对比表

| 产品 | 免费版 | 入门付费 | 专业版 | 企业版 |
|------|-------|---------|-------|-------|
| **PromptPerfect** | ✅ | $8.33/月起 | $20/月 (Standard) | $99.99/月 或自定义 |
| **PromptBase** | ✅ | 平台费20%（卖家付费） | - | - |
| **PromptLayer** | ✅ (基础功能) | Pro (按量) | Team (按量) | Enterprise (自托管) |
| **AIPRM** | ✅ (限制多) | Plus $10/月 | Pro $33/月 | 可叠加购买 |

### 2.2 详细定价

#### PromptPerfect
- **Free**: 基础使用
- **付费计划**: 起价$8.33/月，最高$99.99/月
- **Standard Plan**: $20/月，适合日常用户
- **Interactive Optimizer**: 先进的交互式优化功能

#### PromptBase
- **平台费**: 卖家获得80%收入，平台收取20%
- **价格限制**: 新卖家最高$4.99，待建立销售记录后解除
- **使用场景**: 一次性购买或App使用积分

#### PromptLayer
- **Free**: 基础功能
- **Pro/Team**: 按请求数(transaction)计费，超出后额外收费
- **Enterprise**: 可选GCP/AWS/Azure自托管，或云端单租户
- **合规选项**: HIPAA BAA仅限企业版

#### AIPRM
- **免费版**: 限制多，很多功能不可用
- **Plus ($10/月)**: 基础私有模板和列表
- **Pro ($33/月)**: 完整功能，5个私有列表，可自定义Power Continue
- **叠加购买**: 可购买多份叠加配额

## 三、用户评价

### 3.1 评价来源与评分

| 产品 | 评分来源 | 评分 | 评价数量 | 评价倾向 |
|------|---------|------|---------|---------|
| **PromptBase** | 自有平台 | 4.9/5 | 38,000+ | 正面（但Reddit有争议） |
| **AIPRM** | Chrome Web Store | 3.9/5 | 3,100+ | 中等偏正面 |
| **PromptLayer** | G2 | 较好 | 有限 | 正面（企业用户为主） |
| **PromptPerfect** | 社区 | 较好 | 有限 | 正面 |

### 3.2 用户反馈总结

#### PromptBase 正面评价
- 拥有大量高质量Prompt（200,000+）
- 简单易用的界面
- 社区活跃度高

#### PromptBase 负面评价 (Reddit)
- "根本不值钱，大家不想再付钱买Prompt了"
- 新卖家难以获得销售（"7个 approved MJ prompts，两周零销售"）
- 有人认为这是"噱头和敛财"
- 平台被认为会"走下坡路"

#### AIPRM 负面评价
- "对很多'功能'设置了任意限制"
- 免费版限制过多
- 用户自行开发替代品

#### PromptLayer 正面评价 (G2)
- "帮助我搜索和探索请求历史"
- 适合团队协作
- 强大的版本控制和评估功能

## 四、优势与不足分析

### 4.1 PromptPerfect

**优势**
- 快速优化（10秒内）
- 知名AI公司Jina AI背书
- 支持多种主流模型
- 交互式优化器体验好

**不足**
- 偏向个人用户，团队协作弱
- 无版本控制/评估功能
- 市场认知度有限

### 4.2 PromptBase

**优势**
- 最大的Prompt交易市场
- 200,000+ Prompt库存
- 明确的商业模式（交易抽成）
- 用户基础大（425,000+）

**不足**
- 买卖模式的价值受质疑
- 新卖家难以起步
- 无协作/版本管理功能
- Reddit负面舆论较多

### 4.3 PromptLayer

**优势**
- 最完善的企业级功能
- 强大的版本控制和评估体系
- 多LLM提供商支持
- 丰富的客户案例（Gorgias, NoRedInk等）
- 完善的安全合规（SOC2, HIPAA等）

**不足**
- 价格相对较高
- 主要面向技术团队
- 中小企业门槛高

### 4.4 AIPRM

**优势**
- 零门槛（浏览器扩展）
- 社区模板丰富
- 与ChatGPT/Claude深度集成

**不足**
- 限制过多（免费版）
- 功能相对基础
- 依赖于第三方平台（ChatGPT/Claude）

## 五、差异化机会

### 5.1 市场竞争空白

基于以上分析，Prompt Vault 可考虑以下差异化方向：

| 方向 | 机会说明 | 目标用户 |
|------|---------|---------|
| **个人知识管理** | 专注个人用户的Prompt收藏、分类、复用 | 个人开发者、内容创作者 |
| **中文本地化** | 目前竞品多为英文，中文用户体验差 | 中文用户群体 |
| **轻量级协作** | 比PromptLayer轻量，比PromptBase更重协作 | 小团队/工作室 |
| **Prompt评估** | 简化版Evals功能，降低企业级功能门槛 | 中小企业 |
| **社交发现** | 社区发现、评价、讨论，而非交易 | 学习者、爱好者 |

### 5.2 具体建议

1. **聚焦个人知识管理场景**
   - 竞品多聚焦于企业市场或交易市场
   - 个人用户需要一个好用的"第二大脑"

2. **强化中文生态**
   - 目前主要竞品都是英文为主
   - 中文Prompt市场存在空白

3. **轻量团队协作**
   - PromptLayer太重，AIPRM太轻
   - 2-10人小团队是蓝海

4. **智能化辅助**
   - 参考PromptPerfect的优化能力
   - 但专注于"我的Prompt优化"而非通用优化

5. **开放生态**
   - 提供API/导出功能
   - 避免像AIPRM那样被平台绑定

## 六、总结

| 竞品 | 核心优势 | 核心劣势 | 适合场景 |
|------|---------|---------|---------|
| PromptPerfect | 快速优化 | 无协作/版本 | 个人优化 |
| PromptBase | 市场规模 | 价值受质疑 | 购买/出售 |
| PromptLayer | 企业级完整方案 | 门槛高 | 大企业 |
| AIPRM | 零门槛集成 | 限制多 | 个人尝鲜 |

**Prompt Vault 差异化定位建议**: 面向中文个人/小团队用户的轻量级Prompt知识管理系统，兼顾易用性与协作性。
