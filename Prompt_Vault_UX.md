## 1. 整体布局与导航结构

### 1.1 布局架构

采用 **顶部导航栏 + 左侧边栏 + 主内容区** 的经典三栏布局，兼顾功能性与简洁性。

```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo]  Prompt Vault    [🔍 搜索 (⌘K)]   [+ 新建]  [👤 头像]  │  ← 顶部导航 (56px)
├────────────┬────────────────────────────────────────────────────┤
│            │                                                    │
│  📁 全部    │                                                    │
│  ⭐ 收藏    │                                                    │
│  🕐 最近    │                   主内容区                         │
│            │                                                    │
│  ───────── │                                                    │
│  📂 文件夹  │                                                    │
│  ├ 写作    │                                                    │
│  ├ 编程    │                                                    │
│  └ 营销    │                                                    │
│            │                                                    │
│  ───────── │                                                    │
│  🏷️ 标签   │                                                    │
│  ├ prompt  │                                                    │
│  └ template│                                                    │
│            │                                                    │
├────────────┴────────────────────────────────────────────────────┤
│  [⚙️ 设置]  [? 帮助]                              v1.0         │  ← 底部状态栏 (32px)
└─────────────────────────────────────────────────────────────────┘
   侧边栏         │
   (240px)       │
                 │
```

**布局参数**：
- 顶部导航高度：56px
- 左侧边栏宽度：240px（可折叠至 64px 图标模式）
- 主内容区：自适应（最小 600px）
- 底部状态栏高度：32px（可选，桌面端显示）

### 1.2 导航结构

| 层级 | 入口 | 功能 | 优先级 |
|------|------|------|--------|
| **全局顶部** | Logo | 返回首页 | P0 |
| **全局顶部** | 搜索框 | 全局搜索 | P0 |
| **全局顶部** | 新建按钮 | 创建 Prompt | P0 |
| **全局顶部** | 用户菜单 | 设置/退出 | P0 |
| **左侧边栏** | 快速筛选 | 全部/收藏/最近 | P0 |
| **左侧边栏** | 文件夹树 | 分类导航 | P0 |
| **左侧边栏** | 标签云 | 标签筛选 | P1 |
| **面包屑** | 页面路径 | 层级导航 | P1 |

### 1.3 交互行为

- **侧边栏折叠**：点击左上角图标或 `⌘+\` 切换折叠状态
- **拖拽排序**：文件夹和标签支持拖拽排序
- **快捷键支持**：
  - `⌘K` / `Ctrl+K`：打开搜索
  - `⌘N` / `Ctrl+N`：新建 Prompt
  - `⌘S` / `Ctrl+S`：保存当前编辑
  - `⌘\`：折叠/展开侧边栏
  - `Esc`：关闭弹窗/取消编辑

## 3. 交互流程与关键操作

### 3.1 核心用户流程

#### 流程 1：创建新 Prompt

```
点击 [新建] 
    ↓
选择类型：[空白] / [从模板] / [复制]
    ↓
填写基本信息（标题/描述/文件夹）
    ↓
编辑 Prompt 正文（Markdown）
    ↓
（可选）添加变量和标签
    ↓
保存 → 跳转至详情页
```

**关键操作点**：
- `⌘N` 全局快捷新建
- 模板市场入口在新建弹窗中
- 自动检测 `{{variable}}` 提示添加变量

#### 流程 2：测试 Prompt

```
在详情页点击 [测试面板]
    或
选择 Prompt → [测试] 按钮
    ↓
选择模型（默认上次选择）
    ↓
输入测试内容
    ↓
点击 [运行] 或 ⌘+Enter
    ↓
查看输出结果
    ↓
（可选）保存为新版本
    ↓
（可选）返回编辑优化
```

**关键操作点**：
- 测试面板可拖拽调整宽度
- 支持 `⌘+Enter` 快速运行
- 历史测试记录可追溯

#### 流程 3：A/B 测试对比

```
在测试页切换到 [A/B 对比] 模式
    ↓
选择/编辑 Prompt A
    ↓
选择/编辑 Prompt B
    ↓
输入相同测试内容
    ↓
点击 [同时运行 A 和 B]
    ↓
查看并排输出对比
    ↓
标记优胜者 / 保存最优版本
```

| Danger | 红色填充 | 危险操作（删除） |

**按钮状态**：
- Default → Hover（亮度+10%）→ Active（亮度-10%）→ Disabled（透明度50%）

**输入框**：
- 边框圆角：6px
- 内边距：12px 16px
- 聚焦态：蓝色边框 + 外发光

**卡片**：
- 圆角：12px
- 阴影：`0 1px 3px rgba(0,0,0,0.12), 0 1px 2px rgba(0,0,0,0.24)`
- Hover 阴影：`0 3px 6px rgba(0,0,0,0.16), 0 3px 6px rgba(0,0,0,0.23)`

**标签（Tag）**：
- 圆角：16px（胶囊形）
- 内边距：4px 12px
- 背景：主色 15% 透明度

**弹窗**：
- 圆角：16px
- 遮罩：黑色 50% 透明度
- 最大宽度：560px
- 居中显示

### 4.4 图标风格

- **图标库**：使用 [Lucide Icons](https://lucide.dev/)（一致性高 + 开源免费）
- **尺寸**：16px（正文）/ 20px（标题）/ 24px（装饰）
- **颜色**：继承文字颜色，或使用功能色
- **风格**：2px 描边 + 圆角末端

## 5. 响应式设计要点

### 5.1 断点设计

| 断点 | 屏幕宽度 | 布局变化 |
|------|----------|----------|
| Mobile | < 640px | 隐藏侧边栏，底部导航，堆叠布局 |
| Tablet | 640px - 1024px | 侧边栏可折叠，单列卡片 |
| Desktop | 1024px - 1440px | 标准三栏布局 |
| Wide | > 1440px | 侧边栏加宽，内容区居中（最大 1200px） |

### 5.2 移动端适配要点

**导航变化**：
- 顶部导航简化：Logo + 搜索 + 新建
- 侧边栏改为抽屉式（从左滑出）
- 底部新增 Tab 栏：首页 / 搜索 / 创建 / 设置

**布局调整**：
- 卡片单列显示
- 详情页：编辑区与测试区 Tab 切换（而非并排）
- 测试面板全屏模式

**交互优化**：
- 触摸区域最小 44px
- 支持手势：左滑删除、右滑收藏
- 长按显示上下文菜单

### 5.3 响应式组件策略

| 组件 | 桌面端 | 移动端 |
|------|--------|--------|
| 搜索框 | 顶部导航内置 | 底部浮窗按钮 |
| 筛选栏 | 水平排列 | 折叠为"筛选"按钮 |
| Prompt 卡片 | 显示完整信息 | 仅显示标题+标签 |
| 测试面板 | 左右分栏 | 上下 Tab 切换 |
| 快捷键提示 | 显示快捷键 | 隐藏快捷键 |

### 5.4 适配优先级

1. **P0**：首页可用、列表可浏览、搜索可用、详情可查看
2. **P1**：编辑功能、测试功能、侧边栏导航
3. **P2**：A/B 测试、拖拽交互、快捷键
